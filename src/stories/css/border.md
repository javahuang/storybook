# border

div 的宽度 = border 的宽度(border-width) + content 的宽度(width)


## FAQ

### 子标签的 marginTop 不能撑开父标签的高度

子和父的 margin 发生了折叠，可以设置父元素 `overflow:hidden;` 触发 BFC 解决
